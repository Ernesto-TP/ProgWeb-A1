MeFalta - Qué película o serie me falta ver

El proyecto consiste en el diseño de una aplicación web que permite a los usuarios llevar un registro de las películas y series que han visto y descubrir qué les falta por ver. La aplicación se conecta a APIs de bases de datos cinematográficas para obtener información actualizada sobre estrenos, calificaciones y recomendaciones personalizadas.

Tecnologías

  • HTML5
  • JavaScript
  • FastAPI
  • PostgreSQL
