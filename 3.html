Foto22 - Web para gestión de fotos

El proyecto consiste en el diseño de una plataforma web para la gestión y organización de fotografías digitales. Los usuarios pueden subir, etiquetar, organizar en álbumes y compartir sus fotos de manera segura. La aplicación incluye funcionalidades de búsqueda avanzada, edición básica de imágenes y almacenamiento en la nube.

Tecnologías

  • React
  • Node.js
  • MongoDB
  • AWS S3
